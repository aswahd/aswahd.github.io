GGN Approach

  1. Predict Pairwise affinities

  2. Extract masks from the learned affinities

  3. Rank the extracted masks according to objectness score

  4. Select the highest ranking masks as pseudo ground truth

  5. Add the pseudo ground masks to the curated annotations

  6. Train a class-agnostic model (e.g., class-agnostic Mask R-CNN)

Overview of the approach

Learning pairwise pixel affinities

Affinity: whether two neighboring pixels should be grouped together or not

For, I ∈ ℝ3xHxW, consider its 3x3 immediate neighbors.

The pairwise affinity predictor outputs, P ∈ {0, 1}8xHxW affinity map.

PA Predictor Model:

  • Any dense (pixelwise) prediction network such as FCN, UNet can be used.

Training from non-exhaustive segmentation masks:

  • Most datasets come with non-exhaustive segmentation masks. E.g., COCO

Q. How to learn affinities of an object in the absence of a mask?

  • Ideally if all pixels are labeled, we can consider all pairs of neighboring pixels.
  • However, not every objects/pixels are annotated because exhaustive annotation is expensive.

  • In particular, non-annotated out-of taxonomy objects can’t be distinguished from background.

Solution:
| Only use neighboring pixels with object-object or object–background relations for training pairwise affinities.

| ignore the background-background pairs.

  • Simply put, use the labeled pixels only.

  • Our expectations are pairwise affinities generalize to unseen objects. Indeed, this is the case.

What does a Pairwise Affinity predictor learn?
| Pairwise affinity should discriminate between instance boundaries and instance interiors.

| PA predictor should be robust to boundaries of all objects, not just the categories seen during training

  • Affinity generalizes to unseen objects

| Edge detectors do not learn instance boundaries; affinities do.

| RPN overfits to the training categories.

Class imbalance when training pairwise affinities:

  • Only pixels at an object’s boundary has zero-valued affinities; all other pixels have affinity 1.

  • Use inverse frequency weighting (e.g., 0.05 for positive affinities).

Augmenting with pseudo ground truth

Grouping pixels into regions

One may use

  • Connected Component (CC) algorithm

  • Graph-based hierarchical grouping (GBH)

  • Oriented Watershed Transform (OWT)

  • Globalized contour through Normalized Cut (gPb)

  • Ultrametric Contour Map (UCM)

to cluster affinities into instance masks.

Affinities to masks:

  • Aggregate the image pairwise affinity map into a semantic edge map using pooling along the channel dimension; e.g., reducing from 8-channels to 1-channel

  • Pass the semantic edge map to OWT to generate initial segments

  • Globalize edges using normalized cut

  • Input the average of edge map and the globalized version to UCM

Computing Objectness

| Not all generated masks are appropriate; some are overly-segmented and others are under-segmented.

| Decide which pseudo-GT masks to select for training detectors.

| Generated masks are ranked according to their objectness score.

Objectness score: determine whether there is an object or not inside a proposed bounding box

| Bounding boxes with low objectness score are discarded from further processing.

How to describe objectness score?

| Based on low-level features such as shape and contours

| Or it can be learned, e.g., region proposal network (RPN)

Objectness score from low-level features:

Intuition: Objects with

  • strong inner pairwise affinities

  • and weak affinities at the boundary

should have a high objectness score.

Mathematically,

$$\mathcal{O}_{PA}(R) = \frac{Inner(R)}{R_{inner}} - \frac{Outer(R)}{R_{boundary}}$$ , where

  • Inner(R) - the sum of pairwise affinities of pixels inside R

  • Outer(R) - the sum of pairwise affinities of pixels at the cross-boundaries of R

  • Rinner - the number of pixels inside R

  • Rboundary - the number of pixels on the boundary of R

Generic Grouping Networks (GGNs)

  • generate class-agnostic masks from PA predictor and grouping module

  • use the objectness score to rank regions provided by grouping methods

  • select top ranked regions from each image as pseudo ground-truth (GT) masks

  • train a generic object proposal network

Class agnostic object detection/segmentation

  • GGN can work with any class-agnostic object detection/segmentation model such as Faster R-CNN, Mask R-CNN, and YOLO.

  • Class-agnostic:
    | Convert the multi-class classification prediction head into a binary foreground vs. background classification head

Experiments Evaluation Metrics:

  • mAP: mean average precision over classes averaged over IOU thresholds ranging from .5 to .95 with .05 increments.

  • mAP@0.50IOU: mean average precision at 50% IOU

  • mAP@.75IOU: mean average precision at 75% IOU

  • mAP (small): mean average precision for small objects (area < 322 pixels)

  • mAP (medium): mean average precision for medium sized objects (322 pixels < area < 962 pixels).

  • mAP (large): mean average precision for large objects (962 pixels < area < 100002 pixels).

  • AR@1: average recall with 1 detection

  • AR@10: average recall with 10 detections

  • AR@100: average recall with 100 detections

  • AR@100 (small): average recall for small objects with 100 detections

  • AR@100 (medium): average recall for medium objects with 100 detections

  • AR@100 (large): average recall for large objects with 100 detections
